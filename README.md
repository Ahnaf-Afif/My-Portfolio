# My Portfolio

A modern and responsive personal portfolio website built with **React** and **Vite** to showcase my profile, technical skills, and educational background in a clean and visually engaging way.

This project reflects my journey as an engineering student exploring frontend and full-stack web development. It is designed with a minimal dark-themed interface, a structured layout, and an interactive skills section to create a strong personal web presence.

## Live Demo

[View Live Site](https://ahnaf-afif-portfolio.netlify.app)

## Repository Link

[GitHub Repository](https://github.com/Ahnaf-Afif/My-Portfolio)

## Project Overview

This portfolio website was created to present:

- A brief professional introduction
- Core technical skills and technologies
- Educational background
- A modern and responsive user interface

The main goal of this project is to build a professional portfolio that is simple, visually appealing, and easy to navigate.

## Features

- Responsive design for different screen sizes
- Clean dark-themed UI with glowing accent styling
- Interactive skills section with circular visual arrangement
- Personal introduction section
- Education information section
- Social and contact icons in the navigation bar
- Component-based project structure for better organization and scalability

## How It Works

The application is divided into reusable React components for a cleaner and more maintainable structure:

- **Navbar** – Contains branding and social/contact icons
- **LeftSidebar** – Displays a short professional introduction
- **Skills** – Presents technical skills in an engaging visual format around the profile image
- **RightSidebar** – Shows academic and educational information

This modular structure makes the project easier to manage, customize, and extend in the future.

## Installation and Setup

To run this project locally on your machine, follow these steps:

### Prerequisites

Make sure you have installed:

- **Node.js**
- **npm**

### Clone the Repository

```bash
git clone https://github.com/Ahnaf-Afif/My-Portfolio.git
```

### Navigate to the Project Folder

```bash
cd My-Portfolio
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

Then open the local server URL shown in your terminal.

## Available Scripts
